## Commitment

Vishpala is committed to ensuring that this website is accessible to people with disabilities. We target conformance with the Web Content Accessibility Guidelines (WCAG) 2.2 at Level AA.

Accessibility is a core discipline of our studio. We do not treat it as a compliance exercise.

## Current status

This site has been designed and built with accessibility as a primary requirement. The following measures are in place:

- Semantic HTML structure with appropriate landmark regions
- Skip navigation link to bypass the site header
- All images have meaningful alternative text or are marked as decorative
- Colour contrast meets WCAG 2.2 AA requirements for all text
- The site is fully navigable by keyboard
- Focus indicators are visible and meet contrast requirements
- The site has been tested with VoiceOver (macOS and iOS) and NVDA (Windows)
- Language is declared in the HTML for both English and French content
- All interactive elements have accessible names

## Known limitations

This site is under active development. We will document any known accessibility issues here as they are identified.

No known issues at time of publication.

## Feedback and contact

If you encounter an accessibility barrier on this site — something that prevents you from accessing content or using a feature — please contact us:

- Email: accessibility@vishpala.com
- We aim to respond to accessibility reports within two business days
- We aim to resolve critical barriers within five business days

## Technical information

This site is built with Hugo, a static site generator. Content is delivered as semantic HTML with CSS. No JavaScript is required for any core function of this site.
