## Compromiso

Vishpala está comprometida a garantizar que este sitio web sea accesible para personas con discapacidades. Apuntamos a la conformidad con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.2 en el Nivel AA.

La accesibilidad es una disciplina central de nuestro estudio. No la tratamos como un ejercicio de cumplimiento normativo.

## Estado actual

Este sitio ha sido diseñado y construido con la accesibilidad como requisito principal. Las siguientes medidas están en vigor:

- Estructura HTML semántica con regiones de referencia apropiadas
- Enlace de salto de navegación para omitir el encabezado del sitio
- Todas las imágenes tienen texto alternativo significativo o están marcadas como decorativas
- El contraste de color cumple los requisitos WCAG 2.2 AA para todo el texto
- El sitio es completamente navegable mediante teclado
- Los indicadores de foco son visibles y cumplen los requisitos de contraste
- El sitio ha sido probado con VoiceOver (macOS e iOS) y NVDA (Windows)
- El idioma está declarado en el HTML tanto para el contenido en inglés como en francés
- Todos los elementos interactivos tienen nombres accesibles

## Limitaciones conocidas

Este sitio está en desarrollo activo. Documentaremos aquí cualquier problema de accesibilidad conocido a medida que se identifique.

No hay problemas conocidos en el momento de la publicación.

## Comentarios y contacto

Si encuentra una barrera de accesibilidad en este sitio — algo que le impida acceder al contenido o usar una función — contáctenos:

- Correo electrónico: accessibility@vishpala.com
- Nuestro objetivo es responder a los informes de accesibilidad en un plazo de dos días hábiles
- Nuestro objetivo es resolver las barreras críticas en un plazo de cinco días hábiles

## Información técnica

Este sitio está construido con Hugo, un generador de sitios estáticos. El contenido se entrega como HTML semántico con CSS. No se requiere JavaScript para ninguna función central de este sitio.
